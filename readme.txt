An easy way to add font awesome icons to your navigation menus.

== Description ==

WP Simple Menu Icons provides an easy way to add icons to your WordPress powered navigation menus with the help of [Font Awesome](https://fontawesome.com/).

For usage instructions please refer to the FAQ section below.

== Frequently Asked Questions ==

= Which Icons sets are supported? =

At the moment only Font Awesome is supported. We plan to add support for more sets in the future.

= How do I add icons? =

Navigate to Appearance â†’ Menus in your WordPress admin panel and select the menu you wish to add icons to.

1. Click on the menu item for which you wish to add an icon.
2. Once the settings are expanded, click on the "Setup icon" button.
3. A window will open allowing you to select the icon.
4. Once you've selected the icon and configured it's settings, press the "Save" button.
5. After you've configured all icons, remember to save your menu by clicking the "Save menu" button.
